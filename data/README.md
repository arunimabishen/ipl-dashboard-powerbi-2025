## 📁 Dataset

The dataset used in this project consists of multiple Excel files containing IPL match and player data.

### 📦 Files Included

- Matches data  
- Player statistics  
- Team information  
- Ball-by-ball data  

All files are provided in a compressed format inside the repository:

📂 `data/ipl_dataset.zip`

---

## ⚠️ Note on Dataset Size

The ball-by-ball dataset is large in size (~53 MB), so all data files have been compressed into a single ZIP file for efficient storage and download.

To use the dataset:

1. Download the ZIP file  
2. Extract it locally  
3. Load the required files into Power BI  

---

## 🔗 Data Usage

This dataset is used for analysis and visualization purposes only, helping generate insights related to player performance, team statistics, and match trends in the IPL.
